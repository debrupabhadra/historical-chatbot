# Historical Figure Chatbot
Updated GitHub Pages deployment.
An AI-powered chatbot that lets students have conversations with famous historical figures.

## Features

- Multiple historical figures
- AI-generated responses
- School-friendly language
- Responsive design
- Hosted with GitHub Pages
- Backend powered by Cloudflare Workers

## Technologies

- HTML
- CSS
- JavaScript
- Cloudflare Workers
- Groq API
